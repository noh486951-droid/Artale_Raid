# Artale Raid Commander Pro

Artale Raid Commander Pro 是一款專為線上遊戲副本與公會活動設計的智慧排程與戰績管理系統。透過結合前端視覺化介面及後端 Google Apps Script 資料庫，提供自動化的團隊協作方案。

## 🌟 版本更新 (V1.5.0)

- **自動登入優化**：系統現在能自動記錄使用者的裝置。非團長身分的隊員在開啟網頁時將自動帶入並登入（Auto-login），大幅改善使用者的體驗！
- **無縫表單發送 (No-Reload Submissions)**：解決了以往提交排程或是掉落物表單後，系統會自動重新整理頁面強制返回登入畫面的問題。現在所有的資料傳輸都自動在背景完成並即時更新視覺，填寫順暢不再卡死。
- **CORS 例外防護**：修正並避開 Google Apps Script 伺服器重新導向所引發的 Cross-Origin 跨網域存取警告卡死問題，使背景通訊更為穩定。

## 🚀 主要功能

1. **智慧行事曆 (Time Matrix)**：以色彩區塊呈現每天每個時段出團配置，團員只需點擊即可完成空檔登錄。
2. **自動配置及戰術判定**：系統會根據所選的難度，自動判定是否「職業配置齊全」、「人數達標」，並能即時給出檢核警告。
3. **掉落物及戰績榜 (Leaderboard)**：團長配發獎勵後，將自動統計每一個成員的參與次數與戰利品獲得率並顯示在排行榜。
4. **Discord 聯動發布**：一鍵將確認的副本時段、參戰名單、與集合語音頻道自動推送至設定好的 Discord 頻道裡。
5. **安全區浮動視窗 (Picture-in-Picture)**：對於如困難拉圖斯等複雜副本，可開啟專用的戰鬥地圖與輔助參考圖浮層，並自動置頂於遊戲畫面之中。

## 🛠️ 技術架構

- **前端**：HTML5, CSS3 (Tailwind CSS), Vanilla JavaScript
- **後端及資料庫**：Google Apps Script (GAS) 連結 Google Sheets
- **工具/函式庫**：FontAwesome (圖示), Canvas-Confetti (動畫特效)
